Даалгавар 5: TODO жагсаалт (Console App)
Гүйцэтгэлийн шаардлагууд
Төслийн зорилго, шаардлагыг тодорхойлох нь
Аливаа програмын хамгийн эхний алхам бол “Яг юуг, яагаад хийх хэрэгтэй вэ?” гэдгийг тодруулах. Гар утас, вэб дээр хэчнээн сайн TODO апп байдаг ч консолд ажилладаг хөнгөн жижигхэн хувилбар байнга хэрэг болдог. Жишээ нь, сервертэйгээ SSH-ээр орох эсвэл GUI байхгүй орчинд ганц хоёр даалгавраа тэмдэглэх шаардлага гарахад энэхүү апп дурсамж шиг тусална. Миний хувьд заримдаа Raspberry Pi-д холбогдоод нэг ажил түр сануулах хэрэг гардаг. Тэр үеэр cloud синк, интернет асаалт гээд бодож суух шаардлагагүй, локал текст файл байхад л хангалттай. Иймээс бидний бүтээх TODO жагсаалт нь доорх гол шаардлагатай.

Нэгдүгээрт, хэрэглэгч хүссэн даалгавраа хурдан нэмэх боломжтой байх. Хоёрдугаарт, аль даалгавар дууссан, алинаа устгах шаардлагатай гэдгийг амархан удирдах шаардлагатай. Эцэст нь өгөгдөл тогтвортой хадгалагдах ёстой. Учир нь програмыг хаагаад нээгээд ирэхэд бүх даалгавар санах ойгоос арчигдсан байвал итгэл даахгүй.
